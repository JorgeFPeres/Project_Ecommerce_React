## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/JorgeFPeres/Project_Ecommerce_React.git && cd Project_Ecommerce_React
```

Follow the steps below

```bash
# Install the dependencies
$ npm install

# Start the project
$ npm start
```
